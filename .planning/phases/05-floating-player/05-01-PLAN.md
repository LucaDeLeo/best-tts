  <files>src/content/floating-player.ts</files>
  <action>
Create a new file `src/content/floating-player.ts` with:

1. **Shadow DOM host element creation:**
   - Create div with id `besttts-floating-player`
   - Use `position: fixed; bottom: 20px; right: 20px; z-index: 2147483647`
   - Set `pointerEvents: 'none'` on host (inner container has `pointer-events: auto`)
   - Attach shadow with `mode: 'closed'` for security

2. **Inline styles (getPlayerStyles function):**
   - `:host { all: initial; }` to reset inherited properties
   - `.player-container`: flex layout, gap 8px, padding 12-16px, white background, border-radius 8px, box-shadow, system-ui font, `pointer-events: auto`
   - `.player-btn`: 36x36px circular buttons, #4a9eff background, white text, cursor pointer
   - `.player-btn:hover`: darker background #3a8eef
   - `.player-btn:focus`: 2px solid outline with 2px offset for accessibility
   - `.player-btn:disabled`: opacity 0.5, cursor not-allowed
   - `.progress-text`: min-width 60px for "X/Y" format
   - `.speed-display`: min-width 48px for "X.Xx" format
   - Dark mode media query: `@media (prefers-color-scheme: dark)` with #2a2a2a background, #e5e5e5 text

3. **Player structure (createPlayerDOM function):**
   - Container div with `role="region"` and `aria-label="Best TTS Audio Player"`
   - `tabIndex="0"` for keyboard focus
   - Skip previous button (label: "Previous sentence")
   - Play/pause button (label: "Play" or "Pause")
   - Skip next button (label: "Next sentence")
   - Stop button (label: "Stop playback")
   - Speed display span
   - Progress display span ("1/10" format)
   - Dismiss button (label: "Dismiss player")

4. **Component state interface:**
   ```typescript
   interface PlayerUIState {
     status: 'idle' | 'generating' | 'playing' | 'paused';
     currentChunk: number;
     totalChunks: number;
     playbackSpeed: number;
   }
   ```

5. **Factory function signature:**
   ```typescript
   export function createFloatingPlayer(): {
     update: (state: PlayerUIState) => void;
     destroy: () => void;
     isVisible: () => boolean;
   }
   ```

6. **Singleton pattern:**
   - Track if player already exists (module-level variable)
   - `createFloatingPlayer` returns existing instance if already created
   - `destroyFloatingPlayer` function removes the player

7. **Initial hidden state:**
   - Player is created with `display: none` on the container
   - Only becomes visible when update() is called with non-idle status
